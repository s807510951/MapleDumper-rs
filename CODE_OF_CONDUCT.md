# Code of Conduct

## Our standard

This project is a welcoming, harassment-free space for everyone, regardless of
experience, identity, or background. We expect participants to be respectful,
assume good faith, and keep discussion focused on the technical work.

Examples of unacceptable behavior include harassment, personal or political
attacks, publishing others' private information, and any conduct that would be
inappropriate in a professional setting.

## Scope

This applies to all project spaces, including issues, pull requests, and any
other channel that represents the project, and when an individual is
representing the project in public.

## Enforcement

Report unacceptable behavior privately to the maintainer through the contact on
the repository profile. Reports are reviewed and handled confidentially.
Maintainers may remove, edit, or reject contributions that violate this code,
and may temporarily or permanently bar any contributor for behavior they deem
inappropriate.

This Code of Conduct is adapted from the Contributor Covenant, version 2.1.
